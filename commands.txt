COMANDI PER LE COPIE SU DOCKER

Copia dei nar per utilizzare i processor di hadoop

docker cp ./nifi-hadoop-libraries-nar-2.4.0.nar nifi-ingestion:/opt/nifi/nifi-current/lib 
docker cp ./nifi-hadoop-nar-2.4.0.nar nifi-ingestion:/opt/nifi/nifi-current/lib

Copia del file template per NiFi

docker cp ./flow.json.gz nifi-ingestion:/opt/nifi/nifi-current/conf

Esempio di copia del dataset da locale al container, deve essere eseguito nella dir del dataset

docker cp ./IT nifi-ingestion:/opt/nifi/nifi-current/dataset

Copia del template nella directory del progetto

docker cp nifi-ingestion:/opt/nifi/nifi-current/conf/flow.json.gz /home/alessandro/Documenti/GitHub/BatchProcessing
